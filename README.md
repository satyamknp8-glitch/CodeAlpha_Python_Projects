# CodeAlpha_Python_Projects

Data Science Internship tasks completed for **CodeAlpha**.

## 📁 Projects

### ✅ Task 1: Iris Flower Classification
Classifies Iris flowers (setosa, versicolor, virginica) using measurements, with 5 ML models compared (Logistic Regression, Decision Tree, Random Forest, SVM, KNN). Best model: **SVM — 100% accuracy** on the test set.

📂 [`Task1_Iris_Classification/`](./Task1_Iris_Classification)

### ✅ Task 2: Unemployment Analysis with Python
Analyzes unemployment rate trends in India, investigates the impact of COVID-19 (unemployment jumped from **~7.6% to ~28%** during the Apr–Jun 2020 lockdown), and identifies seasonal patterns and state-wise impact.

📂 [`Task2_Unemployment_Analysis/`](./Task2_Unemployment_Analysis)

## 🛠️ Tools Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
